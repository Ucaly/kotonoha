AstroはゼロJSを基本とする静的サイトジェネレーターです。

Astro is a static site generator that ships zero JavaScript by default — perfect for content-focused sites like blogs.

## Why Astro?

- **Fast by default** — only sends JS when you explicitly add it
- **MDX support** — write Markdown with React-like components
- **Content Collections** — type-safe frontmatter validation

## Hello World

```astro
---
// src/pages/index.astro
const greeting = "Hello, Astro!";
---
<h1>{greeting}</h1>
```

That's all you need to get started.
